# NextUp - Social & Professional Platform

**NextUp** is a full-stack professional networking platform built to help users connect, share updates, and manage their professional profiles. It features real-time connections, a social feed, profile customization, and resume generation.

## 🚀 Features

-   **User Authentication**: Secure login and registration using JWT.
-   **Professional Profiles**: 
    -   Customizable Bio, Work History, and Education.
    -   Profile Picture & Background Banner uploads.
    -   **Downloadable Resume**: Auto-generate a PDF resume from profile data.
-   **Networking**: 
    -   Send and accept connection requests.
    -   View network and pending requests.
-   **Social Feed**: Create posts and comment on updates from your network.
-   **Real-Time Chat**: Instant messaging powered by Socket.io.
-   **Dashboard**: Centralized hub for recent activity and profile stats.

## 🛠️ Tech Stack

### **Frontend**
-   **Framework**: [Next.js](https://nextjs.org/) (React)
-   **State Management**: Redux
-   **Styling**: CSS Modules
-   **HTTP Client**: Axios

### **Backend**
-   **Runtime**: Node.js
-   **Framework**: Express.js
-   **Database**: MongoDB (Mongoose)
-   **Real-time Communication**: Socket.io
-   **File Handling**: Multer (for image uploads)
-   **PDF Generation**: PDFKit
